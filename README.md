# Advent of Code, 2025
### Projects for AOC 2025

#### Requirements
* A toolchain with C++20 support
* [CMake](https://cmake.org/) version 3.18 or higher
* Make

#### Optional
* Python for utility scripts
* [Doxygen](http://www.doxygen.nl/) for documentation
* [Cppcheck](https://github.com/danmar/cppcheck/releases) for static analysis
* ClangFormat for formatting

### Miscellaneous
Binaries, libraries, documentation, logs, etc. are placed in the ``output`` subdirectory.
